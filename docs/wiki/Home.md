# node-tcnet

TCNetは、エンターテイメント業界向けに設計されたオープンプロトコル。
照明、映像、音響などのデバイス間でリアルタイムのタイムコードとメタデータを共有する。

**node-tcnet**は、このTCNetプロトコルのNode.js実装。
